# Basic Prolog Programs

This repository contains a collection of basic Prolog programs that demonstrate classic logic programming problems and techniques.

## Included Programs

- **combination.pl**  
  Generates K-sized combinations from a given list.

- **findpath.pl**  
  Finds paths between nodes in an undirected weighted graph and computes total path cost.

- **fullWords.pl**  
  Converts a non-negative integer into hyphen-separated digit words.

- **goldbach.pl**  
  Finds two prime numbers whose sum equals a given even number greater than 2.

- **map.pl**  
  Solves a map-coloring problem so that adjacent regions do not share the same color.

- **nQueens.pl**  
  Solves the N-Queens problem using backtracking.

- **verifier.pl**  
  Loads the programs and runs sample test cases.

## Requirements

- SWI-Prolog (recommended)
